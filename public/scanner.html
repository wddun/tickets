Door Display
Select an event to generate a QR code
1
Any phone or tablet camera — open the default camera app, point it at this code, and tap the notification. The display page opens instantly in the browser — no app needed.
2
iOS Tickets app — on the display device open the app, tap the gear icon → Display Setup, set the role to Display, connection to WiFi, tap Start, then scan this code with the in-app scanner that appears.
3
Tip: the display page works great in landscape mode on a tablet mounted sideways at the door. You can also paste the copied link directly into any browser.
